Nuestro Proyecto

ODYM E-Commerce

ODYM es una plataforma de comercio electrónico especializada en equipamiento de gimnasio y productos fitness de alta calidad. Nuestro objetivo es hacer accesible el equipamiento profesional para gimnasios caseros y comerciales.

Desarrollado como proyecto semestral, ODYM integra tecnologías modernas como Node.js, MongoDB, y Stripe para ofrecer una experiencia de compra completa y segura.

Node.js MongoDB Stripe JavaScript Tailwind CSS

Misión

Democratizar el acceso a equipamiento fitness de calidad profesional
